Mark posts as old and display a notification message above the content.

== Description ==

When editing a post, check "Mark as Old" in the metabox labeled "Notilicious" to add a notification to the top of the post.  You can also define a custom message for this specific post. If no custom message is provided, a default date difference message will be shown. You can also choose to display a default message set in Settings > Notilicious.
